## Linux

### 解压缩

zip格式：
``` bash
unzip filename.zip
```

tar格式：
``` bash
tar -xvf filename.tar.xxx  # 其中，xxx是gz、bz2、xz、z等后缀（1.15版之后可自动识别压缩格式）
```

### 磁盘
查看磁盘空间：
``` bash
df -hl
```

## Linux子系统（WSL）

移除子系统中的Windows环境变量（仅当次起效）:
``` bash
PATH=$(echo "$PATH" | sed -e 's/:\/mnt.*//g') # strip out problematic Windows %PATH% imported var
```
